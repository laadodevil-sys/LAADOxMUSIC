# LAADOxMUSIC
#  Telegram VC Music Bot

A powerful Telegram Voice Chat Music Bot built with Pyrogram + PyTgCalls.

##  Features
- Play music in voice chat
- YouTube support
- Pause / Resume / Skip
- Queue system
- 24/7 uptime support

## Commands
- /play song name
- /pause
- /resume
- /skip
- /end

## Requirements
- Python 3.10+
- FFmpeg
- Telegram API ID & HASH
- Bot Token
